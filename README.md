# 🦀 Learning Rust

A personal repository where I document my Rust learning journey — chapter by chapter, concept by concept.

## 📂 Structure

| Chapter | Topic |
|---------|-------|
| `chapter1/hello_cargo` | Getting started — Hello World & Cargo basics |
| `chapter2/guess_game` | Building a guessing game |
| `chapter3` | Common programming concepts (variables, data types, functions, control flow) |
| `chapter4/ownership` | Ownership — Rust's core memory model |
| `chapter5/ref_and_borrow` | References & Borrowing |
| `chapter6/slicing` | Slices |
| `chapter7` | Structs, enums, and modules |

## 🚀 Running Any Example

Make sure you have [Rust installed](https://www.rust-lang.org/tools/install), then:

```bash
cd chapterX/<project_name>
cargo run
```

## 🛠 Prerequisites

- [Rust & Cargo](https://www.rust-lang.org/tools/install) — install via `rustup`

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
